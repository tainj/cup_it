# Role
Ты Senior Frontend Developer и UX/UI Designer с опытом работы в финтехе. Твоя задача — полностью переработать фронтенд мобильного банковского приложения, сделав акцент на визуальной привлекательности, персонализации и удобстве пользовательских сценариев.

# Context
Это проект для кейс-чемпионата OTP Bank Changellenge>> Cup. Основная проблема текущих банковских приложений — перегруженность интерфейса и однотипность. 

**Ключевые требования кейса:**
1. Группировка продуктов по жизненным сценариям (не по типам продуктов)
2. Гиперперсонализация контента
3. Упрощённая навигация
4. Визуализация финансовых данных (доходы/расходы)
5. Измеримая бизнес-ценность (конверсия, вовлечённость)

# Tech Stack
- React 18+ с TypeScript
- Tailwind CSS для стилизации
- Recharts или Chart.js для графиков
- Framer Motion для анимаций
- React Context или Zustand для состояния
- Axios для API-запросов

# Requirements

## 1. 🎨 Визуальный дизайн (UI)

### Цветовая схема
- Создай **3 темы на выбор пользователя**:
  - **Light** (светлая): белый фон, тёмный текст, акценты #FF6B35 (оранжевый ОТП)
  - **Dark** (тёмная): тёмно-серый #1A1A2E, светлый текст, акценты #FF6B35
  - **OTP Brand** (фирменная): градиенты оранжевого, белый текст на оранжевом фоне

- Переключатель тем: иконка 🌙/☀️ в хедере, сохранение выбора в localStorage

### Типографика
- Шрифт: Inter или Roboto (системный стек)
- Иерархия: заголовки 24-32px, подзаголовки 18px, текст 14-16px
- Line-height: 1.5 для читаемости

### Компоненты
- **Карточки продуктов**: скругление 16px, тень при наведении, микро-анимации
- **Кнопки CTA**: крупные, контрастные, с иконками
- **Иконки**: lucide-react или heroicons для сценариев

## 3. 📊 Экран аналитики (Analytics Dashboard)

### Графики (Recharts)
1. **Доходы vs Расходы** (столбчатая диаграмма за месяц)
   - Ось X: дни месяца
   - Ось Y: сумма в ₽
   - Два цвета: зелёный (доходы), красный (расходы)

2. **Категории расходов** (Pie Chart)
   - Сегменты: еда, транспорт, развлечения, ЖКХ, прочее
   - При клике на сегмент — детализация

3. **Динамика накоплений** (Line Chart)
   - Тренд за 3/6/12 месяцев
   - Цель (горизонтальная линия)

4. **Прогноз расходов** (Area Chart с градиентом)
   - На основе истории транзакций

### Фильтры
- Период: неделя / месяц / квартал / год
- Категории: чекбоксы для фильтрации
- Экспорт: кнопка "Скачать отчёт" (CSV/PDF)

### Инсайты (AI-like блок)
```
💡 Инсайты для вас:
• В этом месяце вы потратили на 15% меньше на еду
• Если откладывать 5000 ₽/мес, через год будет 60 000 ₽
• Вы часто платите за подписки — проверьте, все ли нужны
```

## 7. 🔌 Интеграция с бэкендом

### API Endpoints (ожидается от Go-бэкенда)
```typescript
// GET /api/v1/showcase?user_id=1
interface ShowcaseResponse {
  user: { name: string; balance: number };
  scenarios: Array<{
    id: number;
    name: string;
    icon: string;
    products: Array<{
      id: number;
      name: string;
      description: string;
      cta: string;
      interest_rate?: number;
    }>;
  }>;
  analytics: {
    income: Array<{ date: string; amount: number }>;
    expenses: Array<{ date: string; amount: number }>;
    categories: Array<{ name: string; amount: number; percent: number }>;
  };
}

// POST /api/v1/analytics/event
interface AnalyticsEvent {
  user_id: number;
  event_type: 'click' | 'view' | 'convert';
  product_id?: number;
  scenario_id?: number;
  timestamp: string;
}
```
